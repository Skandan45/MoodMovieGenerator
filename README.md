# 🎬 Mood-Based Movie Recommendation System

A Flask-based web application that recommends movies according to the user's mood. The system analyzes the user's emotion and provides personalized movie suggestions with trailers.

## 🚀 Features

- 🎭 Mood-based movie recommendations
- 🧠 Emotion detection from user input
- 🎥 Movie trailers integration
- 🔐 User login and authentication
- 🌎 Support for multiple movie languages
- 📱 Responsive user interface

## 🛠️ Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- Bootstrap
- TMDB API

## ▶️ How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/Skandan45/MoodMovieGenerator.git
```

2. Move into the project folder

```bash
cd MoodMovieGenerator
```

3. Install required packages

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit:

```bash
http://127.0.0.1:5000
```
