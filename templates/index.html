System Architecture & NLP Workflow

Understanding the Information Science and engineering modules powering MOODFLIX.

NLP Sentiment Engine

Accepts free-text prompts from users. The text undergoes tokenization and polarity checking using TextBlob to calculate emotional metrics.

Flask Backend Router

A Python Flask server acts as the middleware router. It processes user credentials, manages secure sessions, and interacts with SQLite databases.

TMDB Integration Client

Dispatches secure API calls to TMDB's remote servers. Fetches genre matrices, search results, and embeds YouTube trailer keys dynamically.
